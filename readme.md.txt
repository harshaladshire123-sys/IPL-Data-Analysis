# IPL Cricket Data Analysis

This project analyzes IPL cricket data using Python, Pandas, and Matplotlib.

## Project Overview
The objective of this project is to analyze IPL match and player performance data to gain useful insights such as top run scorers, winning teams, and toss decision trends.

## Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset Files
- matches.csv
- deliveries.csv

## Analysis Performed
- Top 10 IPL Run Scorers
- Most Successful Teams
- Toss Decision Analysis
- Highest Team Scores

## Project Features
- Data Cleaning
- Data Analysis
- Data Visualization
- Graph Generation

## Conclusion
This project helped in understanding data analysis, visualization, and cricket statistics using Python.
